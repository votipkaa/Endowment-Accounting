Data Import

Batch import donors and historical gifts from CSV files to onboard an existing foundation.

Import Donors

Upload a CSV file with donor information — names, addresses, contact details, and donor types. Duplicates (matching display name) are automatically skipped.

Required columns:
display_name (or first_name + last_name)
Optional columns:
donor_type, organization, email, phone, address_line1, city, state, zip_code, country, notes
Import Donors Template
Import Gifts / Contributions

Upload a CSV file with historical gift records. Donors not already in the system will be auto-created. You can specify which fund each gift belongs to in the CSV, or select a default fund for all rows.

Required columns:
amount, contribution_date (or date)
Optional columns:
donor_name, fund_name, gift_type, notes
Import Gifts Template
Onboarding a New Foundation

There are two ways to establish a starting point for each fund:

Option A: Detailed Import
  1. Import donors via CSV
  2. Import all historical gifts via CSV
  3. Set the beginning earnings on each fund to match the investment account
Option B: Summary Starting Point
  1. Create each fund manually
  2. Enter the Beginning Corpus and Beginning Earnings on the fund setup form
  3. This sets the starting value that ties to the investment statements
